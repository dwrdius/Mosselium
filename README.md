# Mosselium - Visualizer for MOSS (Measure of Software Similarity)

Load in a folder containing `readable.html` and its corresponding `matchXX.html` files.

Mosselium will generate a graph where links are *matches* and nodes are users.
- Left click on node: a user will highlight its nearby matches
- Ctrl/Cmd + left click on node: select multiple and see their matches
- Left click on link: open match file
- Shift + hover over (active) link: show preview
- Shift + left click on (active) link: pin preview
- Left click on background: clear selection

The dropdown menu for "Jump To Match" contains the original Moss entries sorted in descending order by match%.
